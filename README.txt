== Description ==

Amazon UK Banner Plugin for WordPress by iWebX

This simple plugin adds an Amazon UK banner underneath the content of every post.
The banner automatically updates from Amazon for latest promotions and offers.
Has settings page where you can pick banner size and add your Amazon Affiliates ID.
100% your links, you earn all money generated from your banners.

Your Amazon ID and your choice of banner size is stored in your Wordpress database so that settings are not lost when upgrading the plugin.

== Installation ==

1. Upload `Amazon-UK-Banner-Wordpress-Plugin-1-1.zip` to the `Upload Plugin` menu in Wordpress.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Enter your Amazon ID and choose your banner size in the 'Banner Settings' page in Wordpress.
